A aplicação pokemonsearch utiliza a cidade fornecida na barra de busca para retornar a temperatura da cidade e um pokemon aleatório, sendo seu tipo uma função da temperatura dessa cidade. Vale ressaltar que caso esteja chovendo, o tipo de pokemon retornado será de raio, independente da temperatura. A API openweathermap serve para obter a temperatura e o clima da cidade, enquanto que a API pokeapi usa determinada regra de temperatura e clima como dados de entrada para retornar o pokemon.
